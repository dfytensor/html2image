# Visual Styles

Named visual identities for social media carousel content.

---

## Style 1: Warm Lifestyle (Default)

| Property | Value |
|---|---|
| Background | `#FFF8F0` (warm cream) |
| Text | `#2D2016` (dark brown) |
| Accent | `#E8833A` (warm orange) |
| Cards | `#FFFFFF` with subtle shadow |
| Muted | `#8B7355` (warm gray-brown) |
| Decorative | Soft blobs, wavy lines, leaf motifs |
| Best for | Food, wellness, home, lifestyle, personal growth |

---

## Style 2: Clean Knowledge

| Property | Value |
|---|---|
| Background | `#F8F9FA` (light cool gray) |
| Text | `#1A1A2E` (dark navy) |
| Accent | `#0984E3` (blue) |
| Cards | `#FFFFFF` with border `#E2E8F0` |
| Muted | `#6C7A89` |
| Decorative | Geometric lines, dots, grid patterns |
| Best for | Education, science, tech tips, tutorials |

---

## Style 3: Bold Social

| Property | Value |
|---|---|
| Background | `#FF6B35` (vibrant orange) or `#1A1A2E` (dark navy) |
| Text | `#FFFFFF` (on dark) or `#1A1A2E` (on light) |
| Accent | `#FFD23F` (bright yellow) or `#FF6B35` (orange) |
| Cards | Semi-transparent white or dark |
| Muted | `#AAAAAA` |
| Decorative | Bold shapes, arrows, exclamation marks |
| Best for | Fitness, motivation, bold statements, challenges |

---

## Style 4: Pastel Soft

| Property | Value |
|---|---|
| Background | `#FDF2F8` (soft pink) |
| Text | `#4A3040` (dark plum) |
| Accent | `#F472B6` (pink) or `#A78BFA` (lavender) |
| Cards | `#FFFFFF` with rounded corners, soft shadow |
| Muted | `#9CA3AF` |
| Decorative | Circles, soft waves, flower line art |
| Best for | Beauty, fashion, baby, gentle topics, romance |

---

## Style 5: Dark Aesthetic

| Property | Value |
|---|---|
| Background | `#0F0F0F` (near-black) or `#1A1A2E` (dark navy) |
| Text | `#F5F5F5` (off-white) |
| Accent | `#E94560` (crimson) or `#FFC300` (gold) |
| Cards | `#1E1E1E` with subtle border glow |
| Muted | `#888888` |
| Decorative | Gradient blobs, grain texture, minimal lines |
| Best for | Luxury, mood, premium, night, photography |

---

## Style 6: Nature Fresh

| Property | Value |
|---|---|
| Background | `#F0FFF4` (mint white) |
| Text | `#1B4332` (forest green) |
| Accent | `#2D6A4F` (green) or `#95D5B2` (sage) |
| Cards | `#FFFFFF` with green-tinted border |
| Muted | `#6B7C6E` |
| Decorative | Leaf shapes, organic curves, botanical line art |
| Best for | Outdoors, sustainability, health, gardening, organic |

---

## Style 7: Minimal Mono

| Property | Value |
|---|---|
| Background | `#FFFFFF` (white) or `#111111` (black) |
| Text | `#111111` (on white) or `#FFFFFF` (on black) |
| Accent | `#333333` or `#CCCCCC` |
| Cards | Border-only, no fill |
| Muted | `#999999` |
| Decorative | Hairline rules, dots, minimal shapes |
| Best for | Quotes, typography, editorial, premium minimal |

---

## Style Selection Matrix

| Content Type | Recommended Style |
|---|---|
| Food / cooking / recipe | Warm Lifestyle |
| Wellness / self-care / sleep | Warm Lifestyle |
| Home / decor / organization | Warm Lifestyle |
| Education / science / tech | Clean Knowledge |
| Tutorial / how-to / guide | Clean Knowledge |
| Fitness / workout / motivation | Bold Social |
| Challenge / transformation | Bold Social |
| Beauty / skincare / makeup | Pastel Soft |
| Fashion / outfit / baby | Pastel Soft |
| Luxury / premium / mood | Dark Aesthetic |
| Photography / night / cinematic | Dark Aesthetic |
| Outdoors / sustainability / health | Nature Fresh |
| Gardening / eco / organic | Nature Fresh |
| Quotes / poetry / editorial | Minimal Mono |
| Typography-focused / artsy | Minimal Mono |
